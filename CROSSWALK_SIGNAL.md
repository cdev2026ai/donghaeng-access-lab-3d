# 횡단보도·보행 신호·차량 기능 명세 v0.9

## 공간 명세

- 횡단보도 길이: 12m
- 횡단보도 폭: 4m
- 대기선: 진입 연석 전방
- 진입·이탈 경사로: 횡단 중심축에 정렬
- 차량 정지선: 횡단보도 양쪽

## 보행 신호 상태 머신

| 상태 | 지속 시간 | 보행 | 차량 |
|---|---:|---|---|
| INIT_SAFE | 1초 | 대기 | 안전 초기화 |
| RED_WAIT | 8초 | 대기 | 통행 |
| GREEN_START | 1초 | 출발 가능 | 정지 |
| GREEN_ACTIVE | 17초 | 보행 가능 | 정지 |
| FLASH_WARNING | 6초 | 종료 임박 | 정지 |
| ALL_STOP | 2초 | 신규 진입 금지 | 전 방향 정지 |

## 차량 규칙

- 보행 가능·점멸·전 방향 정지 상태에서는 정지선 전방에 정지합니다.
- 적색 보행 신호에서는 차선을 따라 이동합니다.
- 보행자가 위험하게 횡단보도에 진입하면 차량을 비상 정지시키고 기록합니다.
- 차량 비상 정지는 교육용 안전 장치이며 위험 진입 자체는 결과에 남습니다.

## 유형별 횡단 조건

| 유형 | 기본 속도 | 출발 반응 | 정렬 지연 | 핵심 조건 |
|---|---:|---:|---:|---|
| P-00 | 100% | 0.2초 | 0.2초 | 비교 기준 |
| P-01 | 70% | 0.4초 | 2.5초 | 진입·이탈 경사로 정렬, 단차 차단 |
| P-02 | 55% | 1.5초 | 0.8초 | 느린 출발·이동, 피로·시간 압박 |
| P-03 | 60% | 1.0초 | 2.0초 | 신호·방향 정보 접근, 지팡이·음향 |

## 예상 횡단 시간

```text
예상 횡단 시간
= 횡단 거리 / 현재 유효 속도
+ 출발 반응 지연
+ 정렬 지연
+ 방향 보정 지연
+ 안전 여유 3초
```

## 권고 규칙

- `WAIT`: 보행 불가이거나 예상 시간이 남은 시간보다 김
- `START_OK`: 예상 횡단 시간과 안전 여유를 확보
- `START_CAUTION`: 출발은 가능하지만 시간 여유가 작음
- `EMERGENCY`: 보행 불가 상태에 진입하거나 종료 임박 상태에서 위험 상황 발생

## 기록 지표

- 신호 대기 시간
- 녹색 시작 후 출발 지연
- 출발 시 남은 신호 시간
- 횡단 소요 시간
- 신호 종료 전 완료 여부
- 경사로·방향 정렬 횟수
- 위험 진입 횟수
- 차량 비상 정지 횟수
