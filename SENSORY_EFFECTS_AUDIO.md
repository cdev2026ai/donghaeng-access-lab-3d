# v0.9.2 감각 효과 보완 명세

## 목적
v0.9.1의 연속 체험·횡단보도·이동 보조 앱 구조를 유지하면서, 이동약자 유형별 감각 차이를 더 분명하게 체험하도록 보완한다.

## 반영 사항

### 1. 시각장애인 시야 제한 강화
- 시야 효과가 실제 화면에 반영되도록 `--vision-opacity`, `--vision-center-radius`, `--vision-mid-radius`, `--scene-vision-blur` 등 CSS 변수 연결을 수정했다.
- 기본 시야 효과 강도를 `중간`에서 `강함`으로 변경했다.
- 설정에 `매우 강함` 옵션을 추가했다.
- P-03의 초기 `visionClarity`와 `directionConfidence`를 낮춰 시작 시점부터 정보 접근 제한이 체감되도록 했다.
- O-03, O-04, O-05 구간에서는 주변부 제한이 조금 더 강화된다.
- 화면 전체를 완전히 검게 만들지 않고 중앙 시야와 흰지팡이 탐지는 유지한다.

### 2. 시야 효과 옵션 차이 명확화
- `낮음`: 주변부 제한을 인식할 수 있는 수준
- `중간`: 중앙 시야만 명확하고 주변 정보가 빠르게 어두워지는 수준
- `강함`: 기본값. 시각장애인 체험에서 바로 차이가 느껴지는 수준
- `매우 강함`: 강사 시연 또는 비교 체험용. 중심부를 제외한 정보 접근이 크게 제한되는 수준

### 3. 휠체어 요철 진동 강화
- O-04 파손 점자블록·거친 노면 구간에서 휠체어 사용자는 더 큰 상하 진동을 경험한다.
- 기존의 약한 미세 흔들림보다 진폭과 주파수를 높였다.
- 상태 UI의 적용 효과에 `요철 진동` 배지가 표시된다.

### 4. 고령 보행자 음성 안내 볼륨 감소
- P-02 고령 보행자·지팡이 사용자 조건에서는 앱 TTS와 음향신호기 소리가 다른 사용자보다 작게 들린다.
- TTS 속도와 피치를 소폭 낮춰 더 작고 느린 안내처럼 느껴지도록 조정했다.
- 자막과 화면 안내는 그대로 유지한다.

## 검수 기준
1. P-03 선택 후 기본 `강함` 옵션에서 주변부 시야 제한이 바로 보여야 한다.
2. 시야 효과 옵션을 `끄기 → 낮음 → 중간 → 강함 → 매우 강함`으로 바꿨을 때 차이가 단계적으로 보여야 한다.
3. P-01로 O-04 구간을 지나갈 때 화면 진동이 기준 보행자보다 크게 느껴져야 한다.
4. P-02 앱 도움 ON에서 음성 안내가 다른 유형보다 작고 약간 낮게 들려야 한다.
5. 설정에서 체험 효과 또는 화면 흔들림을 끄면 해당 효과가 사라져야 한다.
