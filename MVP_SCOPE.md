# MVP 범위 v0.9

## 제품 정의

동행 Access Lab 3D는 이동약자의 이동 보조 앱을 개발하기 전에 학생이 이동 환경과 정보 부족을 직접 경험하고 앱 요구사항을 발견하는 교육용 사전 시연 환경입니다.

## 포함 범위

- 학교 정문 → 버스정류장 연속 3D 경로
- 장애물 O-01~O-05
- 이동약자 P-00~P-03
- 보행 신호 상태 머신과 차량 흐름
- 유형별 횡단 속도·반응·정렬·정보 접근 조건
- 앱 도움 OFF/ON
- 거리·방향·신호·예상 시간·행동 권고
- 비모달 자막과 선택적 TTS
- 피로도·불안감·방향 확신도·시간 압박
- 관찰형 결과와 이동 보조 앱 요구사항 요약
- 강사·개발자 요청형 분석 패널

## 제외 범위

- 장애물마다 열리는 정답 선택 모달
- 점수·등급·랭킹 중심 결과
- 성찰 문항 입력
- 교차 체험 결과 비교
- 브라우저 결과 누적 저장
- JSON·CSV 내보내기
- 실제 보행 신호 API 또는 실시간 지도 연동
- 의료적·진단적 장애 재현

## 완료 기준

1. 유형과 앱 도움 조건을 선택한 뒤 연속 경로를 체험할 수 있다.
2. 횡단보도에서 신호와 차량이 기능적으로 상호작용한다.
3. 같은 조건에서 앱 OFF/ON의 정보 차이를 확인할 수 있다.
4. 결과 화면이 앱 요구사항을 워크북 활동으로 연결한다.
5. 학생 기본 체험에서 자동 퀴즈 모달이 발생하지 않는다.
