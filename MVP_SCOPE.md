# MVP 범위 — v0.6.1 Mission & Result

## 포함

- 3D 기본 환경과 1인칭 이동
- 장애물 5종과 이동약자 유형 시스템
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 피로도·불안감·방향 확신도·시간 압박 상태 UI
- 강화된 시각장애인 시야 효과와 흰지팡이
- 유형 선택 후 미션 브리핑
- 실시간 미션 체크리스트
- 학습형 결과 화면과 자동 해석
- 장애물별 이벤트 기록 확인
- 같은 유형 재체험과 다른 유형 체험
- GitHub Pages 빌드·배포 구조

## 제외

- 시뮬레이터 내부 성찰 작성
- 브라우저 결과 누적 저장
- 교차 체험 비교 화면
- JSON·CSV 내보내기
- 서버·DB·사용자 계정
- 교사 대시보드와 학급 통계
- 실시간 다중 사용자
- 온라인 동료 평가 전송

## 완료 기준

1. 유형 선택 후 미션 브리핑을 거쳐 체험이 시작된다.
2. 체험 중 이벤트·안전 판단·목적지 상태를 확인할 수 있다.
3. 완료 후 이동 기록·상태 지표·장애물별 이벤트 결과를 확인할 수 있다.
4. 결과 화면에서 다른 유형 체험, 같은 유형 재체험, 시작 화면 이동이 가능하다.
5. 성찰·비교·저장·내보내기 UI와 localStorage 사용이 없다.
6. 기존 v0.5.2 체험 효과와 v0.4.1 시작 보호 기능이 유지된다.
