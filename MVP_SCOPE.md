# MVP 범위 — v0.4 Event System

## 포함

- 3D 기본 환경과 1인칭 이동
- 장애물 5종
- 비교 기준 및 이동약자 유형 3종
- 유형별 이동·충돌·감속 규칙
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 장애물별 3개 판단 선택지
- 재확인 루프
- 판단 시간·수행 시간·충돌·차단 기록
- 이벤트 단계 HUD와 최종 이벤트 로그
- GitHub Pages 빌드·배포 구조

## 제외

- 보이는 흰지팡이 모델
- 시야·청각·피로·불안 체험 효과
- 서버·DB·사용자 계정
- 실시간 다중 사용자
- 교사 대시보드
- 분석 데이터 외부 전송

## 완료 기준

1. 활성 장애물마다 이벤트가 한 번씩 시작된다.
2. 상황인지 확인 후에만 판단으로 이동한다.
3. 사용자가 판단 선택지를 고르고 직접 수행한다.
4. 통과 결과가 장애물별로 기록된다.
5. 마지막 결과 화면에서 전체 이벤트 기록을 확인할 수 있다.
