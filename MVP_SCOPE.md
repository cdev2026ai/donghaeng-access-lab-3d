# MVP 범위 — v0.6 Mission, Result & Comparison

## 포함

- 3D 기본 환경과 1인칭 이동
- 장애물 5종과 이동약자 유형 시스템
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 피로도·불안감·방향 확신도·시간 압박 상태 UI
- 강화된 시각장애인 시야 효과와 흰지팡이
- 유형 선택 후 미션 브리핑
- 실시간 미션 체크리스트
- 학습형 결과·자동 해석·성찰 질문
- 브라우저 내 최근 8회 결과 저장
- 2~4개 결과 교차 비교
- JSON·CSV·비교 CSV 내보내기
- GitHub Pages 빌드·배포 구조

## 제외

- 서버·DB·사용자 계정
- 교사 대시보드와 학급 통계
- 실시간 다중 사용자
- 온라인 동료 평가 전송
- 음성 합성 및 방향성 3D 오디오

## 완료 기준

1. 유형 선택 후 미션 브리핑을 거쳐 체험이 시작된다.
2. 체험 중 이벤트·안전 판단·목적지 상태를 확인할 수 있다.
3. 완료 후 기록·상태·이벤트·성찰을 한 화면에서 확인한다.
4. 완료 기록이 브라우저에 저장되고 2~4개를 비교할 수 있다.
5. 개별·비교 결과를 CSV 또는 JSON으로 내보낼 수 있다.
6. 기존 v0.5.2 체험 효과와 v0.4.1 시작 보호 기능이 유지된다.
