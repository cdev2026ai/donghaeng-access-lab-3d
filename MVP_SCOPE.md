# v0.2 장애물 구현 범위

## 포함

- 기본 학교 앞 3D 환경
- 1인칭 이동과 기본 건물 충돌
- 장애물 5종의 3D 오브젝트
- 단차·차량·펜스·볼라드·입간판 충돌
- 경사면 높이 및 속도 변화
- 좁은 구간·거친 노면 속도 변화
- 파손 점자블록 구간 화면 흔들림
- 접근·인지·판단·수행·결과 안내
- 장애물 진행 상태·충돌 횟수·결과 화면
- 장애물 개별 ON/OFF

## 제외

- 페르소나별 신체·이동 파라미터
- 이동약자 × 장애물 15개 조합
- 피로도·불안감·방향 확신도
- 판단 선택 UI
- 개발자 콜라이더 시각화
- 데이터 내보내기와 동료 평가
