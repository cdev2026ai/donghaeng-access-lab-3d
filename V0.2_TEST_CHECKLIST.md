# v0.2 수동 검수 체크리스트

## O-01 높은 단차

- [ ] 정면으로 이동하면 통과가 차단된다.
- [ ] 오른쪽 경사 통로로 우회할 수 있다.
- [ ] 충돌 횟수가 과도하게 연속 증가하지 않는다.
- [ ] 통과 후 상태가 `통과`로 바뀐다.

## O-02 가파른 경사

- [ ] 경사 표면에서 카메라 높이가 부드럽게 올라갔다 내려온다.
- [ ] 속도가 평지 대비 약 55%로 감소한다.
- [ ] 좌우 평지로도 우회할 수 있다.

## O-03 불법 주차 차량

- [ ] 차량과 펜스를 뚫고 지나가지 않는다.
- [ ] 오른쪽 좁은 통로를 천천히 통과할 수 있다.
- [ ] 왼쪽 위험 표시 구간으로 우회할 수 있다.

## O-04 파손 점자블록

- [ ] 정상 구간, 단절 구간, 잘못 연결된 구간이 보인다.
- [ ] 거친 구간에서 속도가 감소한다.
- [ ] 화면 흔들림을 설정에서 끌 수 있다.

## O-05 볼라드·입간판

- [ ] 볼라드와 입간판에 충돌한다.
- [ ] 볼라드 사이 간격으로 통과할 수 있다.
- [ ] 막힌 경우 뒤로 이동해 재진입할 수 있다.

## 공통

- [ ] 설정에서 각 장애물을 끄고 켤 수 있다.
- [ ] 꺼진 장애물은 보이지 않고 충돌하지 않는다.
- [ ] 상황인지→상황판단→동작수행→결과 문구가 표시된다.
- [ ] 목적지 도착 시 통과 수, 충돌 수, 이동 차단 수가 표시된다.
- [ ] R 키 재설정 시 장애물 상태와 결과 수치가 초기화된다.
