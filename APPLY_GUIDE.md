# v0.9.4 적용 안내

## 적용 대상

기존 v0.9.3 프로젝트에 v0.9.4 불법 주차 차량 경유 경로 보완을 적용한다.

## 적용 방법

1. 현재 GitHub 저장소 폴더를 연다.
2. `동행_Access_Lab_3D_불법주차경로_덮어쓰기_v0.9.4.zip`의 압축을 푼다.
3. 압축 해제된 폴더 안의 파일과 폴더를 저장소 최상위에 복사한다.
4. 같은 이름의 파일은 덮어쓴다.
5. 아래 명령을 실행한다.

```bash
npm ci
npm run dev
```

- 버스정류장 목적지가 왼쪽 뒤편으로 이동했는지

- 진행 카드에 불법 주차 구간 항목이 표시되는지
