# v0.9.6 적용 가이드

## 중요한 안내

v0.9.5 덮어쓰기본에는 실제 3D 배치 코드인 `src/main.ts`가 빠져 있어, 기존 저장소에 적용해도 볼라드·입간판·점자블록 위치가 바뀌지 않을 수 있었습니다.

v0.9.6 덮어쓰기본에는 `src/main.ts`가 포함되어 있습니다.

## 적용 방법

1. 덮어쓰기본 압축을 풉니다.
2. 압축을 푼 폴더 안의 내용물을 현재 GitHub 저장소 최상위 폴더에 복사합니다.
3. `src/main.ts`가 덮어쓰기되는지 확인합니다.
4. 아래 명령을 실행합니다.

```bash
npm ci
npm run dev
```

## 확인 포인트

- `src/main.ts`가 실제로 교체되었는지 확인
- 화면 상단 버전이 v0.9.6으로 보이는지 확인
- 점자블록이 신호등 이후부터 목적지 방향으로 이어지는지 확인
- 볼라드·입간판이 버스정류장 앞에 있는지 확인
