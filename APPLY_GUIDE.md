# v0.9.3 적용 안내

이 버전은 v0.9.2 감각 효과 보완본 위에 **고령 보행자 전용 약한 희뿌연 시야 효과**를 추가한 패치입니다.

## 적용 방법

1. `동행_Access_Lab_3D_고령자시야효과_덮어쓰기_v0.9.3.zip`의 압축을 풉니다.
2. 압축 안의 내용물을 현재 GitHub 저장소 최상위 폴더에 복사합니다.
3. 파일 교체 메시지가 나오면 `대상 폴더의 파일 덮어쓰기`를 선택합니다.
4. 터미널에서 실행합니다.

```bash
npm ci
npm run dev
```

## 확인 순서

1. P-00 비교 기준 보행자를 선택하고 화면이 선명한지 확인합니다.
2. P-02 고령 보행자·지팡이를 선택하고 화면이 일반인보다 살짝 희뿌옇고 대비가 낮은지 확인합니다.
3. 고령 보행자로 오래 이동하거나 횡단보도에서 시간이 부족해질 때 흐림이 조금 더 증가하는지 확인합니다.
4. P-03 시각장애인 모드는 여전히 주변부 제한이 강하게 적용되는지 확인합니다.
5. 설정에서 체험 효과를 끄면 고령자 시야 효과도 사라지는지 확인합니다.
