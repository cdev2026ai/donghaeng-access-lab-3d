# 체험 관찰 체크리스트

이 체크리스트는 시뮬레이터를 체험하면서 “앱이 어떤 정보를 제공해야 하는가?”를 찾기 위한 기록용 문서입니다.

## 체험 전

- [ ] 이동약자 유형을 확인했다.
- [ ] 앱 도움 OFF 또는 ON 조건을 확인했다.
- [ ] 목적이 빠른 도착이 아니라 정보 부족 발견임을 이해했다.

## 이동 중 관찰

| 구간 | 관찰 질문 | 기록 |
|---|---|---|
| 보도 진입 | 단차와 경사로 위치를 쉽게 알 수 있었나? |  |
| 횡단보도 대기 | 보행 신호와 남은 시간을 확인할 수 있었나? |  |
| 횡단 중 | 현재 속도로 시간 안에 건널 수 있다고 판단했나? |  |
| 불법주차 구간 | 통로 폭과 우회 가능성을 확인할 수 있었나? |  |
| 점자블록 구간 | 유도선이 끊긴 부분에서 방향을 유지할 수 있었나? |  |
| 볼라드 구간 | 목적지 앞 통로 폭과 진입 각도를 확인할 수 있었나? |  |

## 앱 요구사항으로 바꾸기

| 부족했던 정보 | 앱 기능 아이디어 |
|---|---|
| 경사로 위치를 알기 어려움 | 경사로 방향·거리 안내 |
| 남은 신호 시간이 부족함 | 예상 횡단 시간과 대기 권고 |
| 점자블록이 끊김 | 다음 유도선 위치와 방향 안내 |
| 통로 폭이 좁음 | 통과 가능 여부와 우회 안내 |
