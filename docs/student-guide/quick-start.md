# 학생용 빠른 시작

## 접속

- 학생용 URL: https://cdev2026ai.github.io/donghaeng-access-lab-3d/
- 권장 브라우저: Chrome 또는 Edge
- 권장 환경: 키보드와 마우스가 있는 노트북 또는 데스크톱

## 조작법

| 조작 | 기능 |
|---|---|
| W | 앞으로 이동 |
| A / D | 좌우 이동 |
| S | 뒤로 이동 |
| 마우스 | 시점 회전 |
| F | 흰지팡이 탐지, P-03에서 사용 |
| ESC | 일시정지 |
| R | 현재 구간 다시 시작 |

## 추천 체험 순서

1. 이동약자 유형을 선택합니다.
2. 먼저 **앱 도움 없음**으로 체험합니다.
3. 결과 화면에서 부족했던 정보를 확인합니다.
4. 같은 유형으로 **앱 도움 있음**을 켜고 다시 체험합니다.
5. 앱이 제공한 정보가 이동 판단에 어떤 차이를 만들었는지 기록합니다.

## 체험 중 가장 중요한 질문

> 빠르게 도착했는지가 아니라, 이동 중 어떤 정보가 부족했는지 관찰하세요.

## 워크북에 적을 내용

- 어려웠던 구간
- 부족했던 정보
- 앱이 알려주면 좋을 정보
- 안내 방식: 화면, 음성, 진동, 색상, 방향 표시 등
