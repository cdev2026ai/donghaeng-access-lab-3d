# 학생용 체험 가이드

## 체험 목적

이 시뮬레이터는 이동약자 보조 앱 개발 전에 이동 환경을 직접 관찰하기 위한 체험 도구입니다. 목적지까지 빨리 가는 것이 목표가 아니라, 이동 중 필요한 정보를 발견하는 것이 목표입니다.

## 이동약자 유형

| 코드 | 유형 | 관찰 포인트 |
|---|---|---|
| P-00 | 비교 기준 보행자 | 기준 이동 경로와 정보 확인 |
| P-01 | 수동 휠체어 사용자 | 단차, 경사로, 통로 폭, 회전 반경 |
| P-02 | 고령 보행자·지팡이 사용자 | 느린 보행, 신호 시간 부족, 피로, 희뿌연 시야 |
| P-03 | 시각장애인·흰지팡이 사용자 | 시각 신호 접근, 방향 확인, 점자블록 단절, 흰지팡이 탐지 |

## 앱 도움 없음 / 있음

### 앱 도움 없음

환경에 보이는 정보만으로 이동합니다.

- 신호를 직접 확인해야 합니다.
- 경사로 위치를 직접 찾아야 합니다.
- 점자블록 단절이나 볼라드 위치를 직접 파악해야 합니다.
- 현재 속도로 횡단이 가능한지 직접 판단해야 합니다.

### 앱 도움 있음

이동 보조 앱 안내가 나타납니다.

- 신호 상태와 잔여 시간
- 예상 횡단 시간
- 장애물까지의 거리와 방향
- 경사로 또는 우회 경로
- 권장 행동

## 체험 경로

```text
학교 정문
→ 보도 턱·경사로
→ 횡단보도 신호 대기
→ 횡단보도 이동
→ 불법주차 차량 구간
→ 점자블록 단절 구간
→ 볼라드·입간판 구간
→ 버스정류장 도착
```

## 관찰 질문

- 어떤 구간에서 처음 멈추거나 방향을 다시 잡았나요?
- 어떤 정보가 없어서 판단이 어려웠나요?
- 앱이 어떤 정보를 언제 알려주면 좋을까요?
- 화면 안내, 음성 안내, 진동 안내 중 어떤 방식이 적절할까요?
- 같은 길인데 이동약자 유형에 따라 어려움이 어떻게 달라졌나요?

## 결과 화면 읽는 법

결과 화면은 점수가 아닙니다. 다음 정보를 앱 요구사항으로 바꾸기 위한 관찰 기록입니다.

- 신호 대기 시간
- 출발 반응 시간
- 횡단 시간
- 신호 종료 전 완료 여부
- 경로 재조정 횟수
- 이동 차단·충돌
- 부족했던 정보
- 앱이 제공해야 할 정보
