# 프로젝트 기획 개요

## 한 줄 정의

동행 Access Lab 3D는 이동약자 보조 앱 개발 전에 학생들이 이동 장벽과 정보 부족을 직접 관찰하도록 돕는 교육용 3D 체험 시뮬레이터입니다.

## 교육 목표

- 이동약자의 이동을 개인 능력 문제가 아니라 환경과 정보 접근성의 문제로 바라본다.
- 상황인지, 상황판단, 동작수행 단계에서 어디서 어려움이 발생하는지 구분한다.
- 체험에서 발견한 어려움을 이동 보조 앱의 요구사항으로 전환한다.
- 팀 프로젝트에서 구현할 기능의 필요성과 우선순위를 설명할 수 있다.

## 시뮬레이터의 역할

| 구분 | 역할 |
|---|---|
| 3D 시뮬레이터 | 이동 장벽과 정보 부족을 경험하는 사전 체험 환경 |
| 이동 보조 앱 | 거리, 방향, 신호, 위험, 권장 행동을 안내하는 해결 도구 |
| 워크북 | 체험 기록을 문제 정의와 요구사항으로 정리 |
| 팀 프로젝트 | 앱 기능을 설계하고 구현하는 활동 |

## 핵심 경로

```text
학교 정문
→ 보도 턱·경사로
→ 보행 신호 대기
→ 횡단보도
→ 불법주차 차량
→ 점자블록 훼손
→ 볼라드·입간판
→ 버스정류장
```

## 핵심 비교 구조

- 이동약자 유형 비교
- 앱 도움 없음 / 있음 비교
- 환경 정보만 있는 경우와 앱 안내가 있는 경우 비교
- 같은 경로에서 유형별로 달라지는 병목 비교

## 개발 범위

학생용 레포에는 체험과 결과 확인에 필요한 기능만 포함합니다. 체크포인트 이동, 결과 강제 열기, 좌표 확인 같은 검수 도구는 개발자용 별도 레포에서 운영합니다.
