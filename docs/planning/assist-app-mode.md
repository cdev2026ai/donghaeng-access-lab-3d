# 이동 보조 앱 OFF/ON 기획

## 비교 목적

앱 도움 OFF/ON 구조는 학생들이 “앱이 왜 필요한가?”를 직접 확인하도록 만드는 핵심 장치입니다.

## 앱 도움 없음

앱 도움 없음에서는 환경에 존재하는 정보만 사용합니다.

- 신호 잔여 시간 자동 안내 없음
- 경사로 위치 자동 안내 없음
- 장애물 거리·방향 안내 없음
- 예상 횡단 시간 안내 없음
- 점자블록 단절 이후 방향 안내 없음

## 앱 도움 있음

앱 도움 있음에서는 이동 보조 앱이 다음 정보를 제공합니다.

- 신호 상태와 잔여 시간
- 예상 횡단 시간
- 장애물 거리와 방향
- 경사로 위치
- 통로 폭과 우회 가능성
- 권장 행동
- 음성 안내와 화면 안내

## 안내 원칙

- 이동을 멈추는 모달 대신 비모달 패널로 안내합니다.
- 중요한 상황은 짧고 명확하게 표현합니다.
- 사용자 유형별로 필요한 정보를 다르게 제공합니다.
- 정답을 강요하기보다 판단 근거를 제공합니다.

## 결과 화면 연결

결과 화면에서는 다음 두 가지를 함께 보여줍니다.

1. 부족했거나 확인하기 어려웠던 정보
2. 이동 보조 앱이 제공해야 할 정보

학생은 이 내용을 워크북의 앱 요구사항으로 옮깁니다.
