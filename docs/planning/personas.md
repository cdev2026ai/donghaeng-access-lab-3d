# 이동약자 페르소나 정의

## P-00 비교 기준 보행자

| 항목 | 내용 |
|---|---|
| 목적 | 같은 환경의 기준 이동 결과 확인 |
| 주요 특징 | 일반 속도, 일반 시야, 일반 반응 |
| 관찰 포인트 | 기준 조건에서도 환경 정보가 충분한지 확인 |

## P-01 수동 휠체어 사용자

| 항목 | 내용 |
|---|---|
| 핵심 병목 | 동작수행 |
| 주요 제약 | 단차, 경사, 통로 폭, 회전 반경, 경사로 정렬 |
| 체험 효과 | 낮은 시점, 좁은 통로 제약, 요철 진동 |
| 앱 요구사항 | 경사로 위치, 통로 폭, 우회 가능성, 경로 안전성 |

## P-02 고령 보행자·지팡이 사용자

| 항목 | 내용 |
|---|---|
| 핵심 병목 | 상황인지·판단·수행 전반 지연 |
| 주요 제약 | 느린 이동, 출발 반응 지연, 피로 누적, 신호 시간 부족 |
| 체험 효과 | 약한 희뿌연 시야, 낮은 대비, 낮은 음성 안내 볼륨 |
| 앱 요구사항 | 잔여 신호 시간, 예상 횡단 시간, 출발 가능 여부, 휴식·주의 안내 |

## P-03 시각장애인·흰지팡이 사용자

| 항목 | 내용 |
|---|---|
| 핵심 병목 | 상황인지 |
| 주요 제약 | 시각 신호 확인 어려움, 방향 정보 부족, 점자블록 단절 |
| 체험 효과 | 주변부 시야 제한, 흰지팡이 탐지, 방향 확신도 저하 |
| 앱 요구사항 | 신호 상태, 방향, 다음 유도선 위치, 장애물 거리, 음성 안내 |

## 설계 원칙

이동약자 유형은 실제 개인 전체를 대표하지 않습니다. 수업용 비교 체험을 위해 이동 조건과 정보 접근 조건을 단순화한 모델입니다.
