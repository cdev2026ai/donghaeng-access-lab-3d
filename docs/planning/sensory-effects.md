# 감각 효과 설계

## 목적

감각 효과는 특정 장애 경험을 완전히 재현하기 위한 것이 아니라, 정보 접근성과 이동 조건이 달라질 때 체험이 어떻게 달라지는지 관찰하기 위한 교육용 표현이다.

## 유형별 효과

| 유형 | 효과 |
|---|---|
| P-00 비교 기준 | 선명한 시야와 기준 이동 속도 |
| P-01 휠체어 사용자 | 낮은 시점, 단차 차단, 요철 진동 |
| P-02 고령 보행자 | 약한 흐림, 낮은 대비, 작은 음성 안내, 느린 이동 |
| P-03 시각장애인 | 강한 주변부 시야 제한, 흰지팡이 탐지, 신호·방향 정보 제한 |

## 주의 사항

- 체험 효과를 개인 능력 평가처럼 해석하지 않는다.
- 실제 장애 경험 전체를 단순화하지 않는다.
- 앱 도움 ON/OFF 차이를 통해 필요한 정보 제공 방식을 찾는 데 집중한다.
