# 연속 체험 설계

## 설계 의도

학생용 시뮬레이터는 장애물마다 멈춰서 선택지를 고르는 게임형 흐름이 아니라, 실제 이동처럼 연속적으로 길을 따라가며 정보를 관찰하는 구조를 목표로 한다.

## 이동 경로

```text
학교 정문
→ 보도 턱과 경사로
→ 횡단보도 대기선
→ 횡단보도
→ 불법주차 차량 옆 좁은 통로
→ 훼손된 점자블록
→ 목적지 앞 볼라드·입간판
→ 버스정류장
```

## 행동 기록 방식

시뮬레이터는 다음 행동을 내부적으로 기록한다.

- 정지
- 시점 회전
- 경로 이탈과 복귀
- 충돌
- 이동 차단
- 신호 대기
- 출발 지연
- 횡단 완료 여부

이 기록은 결과 화면에서 앱 요구사항을 도출하는 근거로 사용된다.
