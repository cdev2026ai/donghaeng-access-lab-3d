# 횡단보도·신호 기획

## 기획 의도

횡단보도는 이동약자 체험에서 핵심 장면입니다. 신호가 보이더라도 남은 시간, 이동 속도, 출발 지연, 방향 정렬 조건에 따라 안전하게 건널 수 있는지가 달라집니다.

## 신호 상태

```text
적색 대기
→ 보행 신호 시작
→ 보행 가능
→ 종료 임박
→ 전 방향 정지
→ 적색 대기
```

## 유형별 제약

| 유형 | 횡단보도에서의 제약 |
|---|---|
| P-00 | 기준 속도로 신호 내 횡단 가능 |
| P-01 | 경사로 위치와 정렬 시간이 중요 |
| P-02 | 출발 지연과 느린 보행 때문에 시간 내 횡단이 어려움 |
| P-03 | 시각 신호만으로는 보행 가능 여부와 방향 파악이 어려움 |

## 앱이 제공해야 할 정보

- 현재 보행 신호 상태
- 남은 신호 시간
- 현재 속도를 고려한 예상 횡단 시간
- 출발 가능 여부
- 다음 신호 대기 권고
- 횡단 방향
- 반대편 경사로 또는 안전 지점 방향

## 결과 기록

- 신호 대기 시간
- 녹색 신호 후 출발 지연
- 출발 시 잔여 시간
- 실제 횡단 시간
- 신호 종료 전 횡단 완료 여부
- 위험 진입 여부
- 차량 안전 정지 여부
