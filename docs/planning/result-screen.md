# 결과 화면 기획

결과 화면은 점수판이 아니라, 이동 중 부족했던 정보를 정리하는 관찰 화면이다.

## 표시 항목

- 신호 대기 시간
- 출발 반응 시간
- 출발 시 남은 신호 시간
- 횡단 시간
- 신호 종료 전 도착 여부
- 멈춤과 경로 재조정
- 충돌 또는 이동 차단
- 장애물별 관찰 기록
- 부족했던 정보
- 앱이 제공해야 할 정보

## 표현 원칙

| 지양 | 권장 |
|---|---|
| 미션 성공/실패 | 체험 경로 완료 |
| 안전 판단 점수 | 관찰된 행동 기록 |
| 랭킹 | 정보 접근성 차이 |
| 개인 능력 평가 | 환경 조건 분석 |

## 앱 요구사항 연결

결과 화면의 “부족했던 정보”는 학생 프로젝트에서 구현할 앱 기능으로 변환된다.
