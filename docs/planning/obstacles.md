# 환경 장벽과 장애물 정의

## 장애물 목록

| ID | 이름 | 주요 영향 단계 | 설명 |
|---|---|---|---|
| O-01 | 높은 단차·보도 턱 | 동작수행 | 경사로를 찾지 못하면 휠체어 사용자가 이동하기 어려움 |
| O-02 | 가파른 경사로 | 동작수행·피로 | 이동 속도 감소와 피로 누적을 유발 |
| O-03 | 불법주차 차량·좁은 통로 | 상황판단·동작수행 | 보도 폭을 줄이고 우회 판단을 요구 |
| O-04 | 신호등에서 이어지다 훼손된 점자블록 | 상황인지 | 시각장애인의 방향 유지와 정보 접근에 영향 |
| O-05 | 목적지 앞 볼라드·입간판 | 동작수행·진입 판단 | 목적지 진입 직전 통로 폭과 각도를 제한 |

## 이동 메커니즘과의 연결

| 단계 | 의미 | 관련 장애물 |
|---|---|---|
| 상황인지 | 무엇이 있는지 알아차리는 단계 | O-03, O-04, O-05 |
| 상황판단 | 기다릴지, 우회할지, 진행할지 결정하는 단계 | O-02, O-03, 횡단보도 신호 |
| 동작수행 | 실제로 이동하거나 통과하는 단계 | O-01, O-02, O-03, O-05 |

## 앱 요구사항으로의 연결

장애물은 단순히 이름을 알려주는 대상이 아닙니다. 앱은 다음 정보를 함께 제공해야 합니다.

- 장애물까지의 거리
- 장애물 방향
- 통과 가능 여부
- 경사로 또는 우회 경로
- 남은 통로 폭
- 사용자 유형에 따른 위험도
- 권장 행동
