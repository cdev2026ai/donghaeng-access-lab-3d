# v0.9.3 감각 효과·음성 효과 명세

## 목적

이 문서는 이동약자 유형별 감각 효과가 수업용 비교 체험에서 어떻게 다르게 표현되는지 정의합니다. 효과는 특정 집단 전체를 단정하기 위한 것이 아니라, 이동 보조 앱 개발 전 정보 접근 차이를 이해하기 위한 교육용 조건입니다.

## 유형별 감각 효과

### P-00 비교 기준 보행자

- 선명한 시야
- 정상 대비와 채도
- 정상 음성 안내 볼륨
- 요철 진동 없음

### P-01 수동 휠체어 사용자

- 시야 자체는 선명하게 유지
- O-04 파손 점자블록·요철 구간에서 화면 진동 강화
- `화면 흔들림` 옵션을 끄면 요철 진동도 비활성화
- 상태 UI에 `요철 진동` 표시

### P-02 고령 보행자·지팡이 사용자

- 시각장애인처럼 어두워지거나 주변부가 막히지는 않음
- 일반인보다 약한 희뿌연 시야 적용
- 낮은 대비, 낮은 채도, 아주 약한 블러 적용
- 피로도와 시간 압박이 높아질수록 흐림과 대비 저하가 소폭 증가
- 앱 TTS와 음향신호기 소리가 다른 사용자보다 작게 들림
- TTS 속도와 피치를 약간 낮춤
- 상태 UI에 `약한 흐림·대비 저하` 표시

### P-03 시각장애인·흰지팡이 사용자

- 주변부 시야 제한과 중앙 시야 유지
- 시야 효과 강도: 끄기 / 낮음 / 중간 / 강함 / 매우 강함
- O-03, O-04, O-05에서 시야·방향 정보 부담 증가
- 흰지팡이 표시와 F 키 탐지 유지
- 정보 명료도와 방향 확신도 변화가 결과에 반영

## 접근성 안전장치

- 체험 효과 전체 ON/OFF 가능
- 화면 흔들림 ON/OFF 가능
- 고대비 UI 옵션 제공
- 저사양 모드에서는 과도한 CSS backdrop-filter를 제거
- 모든 음성 안내는 화면 텍스트로도 제공
