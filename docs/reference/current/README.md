# 기존 제작 문서 보관

이 폴더는 이번 문서 정리 이전에 레포 루트에 있던 기존 Markdown 문서를 함께 보관한 위치입니다.

## 포함 문서 예시

- `PERSONAS.md`
- `OBSTACLES.md`
- `CROSSWALK_SIGNAL.md`
- `ASSIST_APP.md`
- `PARKED_CAR_ROUTE.md`
- `SENSORY_EFFECTS_AUDIO.md`
- `SIGNAL_OBJECT_ELDERLY.md`
- `BUILD_REPORT.md`
- `CHANGELOG.md`
- `V0.9.x_TEST_CHECKLIST.md`

새로 정리된 문서는 `docs/planning`, `docs/student-guide`, `docs/teacher-guide`, `docs/developer-guide`, `docs/architecture`를 우선 참고하세요.

기존 문서는 구현 과정의 변경 이력과 상세 기록을 확인할 때 사용합니다.
