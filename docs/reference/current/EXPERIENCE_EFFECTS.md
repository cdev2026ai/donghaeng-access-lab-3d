# 체험 효과·상태 UI v0.9

## 상태 축

- 피로도: 이동 시간, 경사, 거친 노면, 차단에 따라 증가
- 불안감: 위험 진입, 차량 접근, 충돌, 신호 종료 임박에 따라 증가
- 방향 확신도: P-03 점자블록 단절·경로 편차에서 감소, 탐지·안내에서 회복
- 시간 압박: 횡단보도 잔여 시간이 줄어들수록 증가
- 시야 명료도: P-03 시야 효과와 정보 조건 반영

## UI 원칙

- 기본 학생 화면에는 간결한 상태 바만 표시합니다.
- 효과가 이동을 강제로 중단하지 않습니다.
- 시야 효과는 완전한 암전보다 정보 접근 제한을 표현합니다.
- 흔들림·시야 강도·고대비·흰지팡이·음향신호·TTS를 설정에서 조절할 수 있습니다.
- 저사양 모드에서도 신호와 앱 텍스트 안내는 유지합니다.

## 신호와 상태 연결

- 종료 임박: 시간 압박·불안감 증가
- 위험 진입: 불안감 급증, 차량 비상 정지
- 다음 신호 대기: 시간 압박 완화
- 앱의 근거 있는 안내: 불안감과 방향 불확실성 완화

## 윤리적 표현

상태 수치는 장애 정도나 개인 능력을 평가하지 않습니다. 동일 환경에서 이동 조건과 정보 지원이 어떤 부담을 만드는지 설명하기 위한 시뮬레이션 값입니다.
