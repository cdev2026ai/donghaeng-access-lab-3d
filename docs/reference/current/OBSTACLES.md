# 장애물·경로 명세 v0.9

장애물은 독립 미션이 아니라 하나의 연속 이동 경로 안에 배치됩니다.

| ID | 장애물 | 경로 장면 | 주요 관찰 |
|---|---|---|---|
| O-01 | 높은 단차·보도 턱 | 학교 정문·횡단 진입 | 단차 통과, 경사로 위치 |
| O-02 | 가파른 경사로 | 보도 진입·이탈 | 속도·피로·경사 부담 |
| O-03 | 불법 주차 차량·좁은 통로 | 횡단 후 보도 | 통과 폭·우회 방향 |
| O-04 | 파손·끊긴 점자블록 | 횡단 후 유도 경로 | 방향 정보 단절 |
| O-05 | 볼라드·입간판 | 버스정류장 접근 | 폭·회전·재진입 |

## 상호작용

- 접근 시 전체 화면 모달을 열지 않습니다.
- HUD와 앱 패널은 짧은 정보만 제공합니다.
- 충돌·차단·정지·경로 조정·통과를 자동 기록합니다.
- 앱 OFF에서는 환경 정보만, 앱 ON에서는 거리·방향·통과 조건·우회 행동을 안내합니다.

## 횡단보도 통합

O-01과 O-02는 횡단보도 진입·이탈 경사로와 연결됩니다. 신호 잔여 시간, 경사로 정렬, 횡단 속도가 함께 작동하므로 단일 장애물 통과가 아니라 횡단 전체 조건으로 평가합니다.
