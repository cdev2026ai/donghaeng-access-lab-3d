# 장면 좌표와 배치 기준

## 좌표 기준

- 사용자는 학교 정문 쪽에서 출발해 z값이 감소하는 방향으로 이동합니다.
- 횡단보도는 `CROSSWALK_CENTER_X = 4.2`를 중심으로 배치되어 있습니다.
- 북측 보도는 도로 건너편 구간이며, z가 음수인 영역입니다.

## 주요 좌표표

| 요소 | 좌표 기준 | 설명 |
|---|---|---|
| 시작점 | x 0 / z 22 | 학교 정문 앞 |
| O-01 보도 턱 | z 17 | 단차와 우측 경사로 |
| O-02 경사로 | x 4.2 / z 11.7 | 가파른 경사 체험 |
| 횡단보도 진입 | z 5.8 | 보행 신호 후 진입 |
| 횡단보도 이탈 | z -5.8 | 도로 건너편 보도 시작 |
| O-03 불법주차 차량 | x 약 2.7 / z 약 -9.7 | 도로 건너편 기준 우측 이동 반영 |
| O-04 점자블록 단절 | x 약 -1.2 / z 약 -16.7 | 신호등 이후 목적지 방향 유도선 단절 |
| O-05 볼라드·입간판 | x 약 -4.0 / z 약 -23.25 | 버스정류장 앞 통로 |
| 목적지 | x -5.8 / z -26.0 | 버스정류장 |
