# 문제 해결 가이드

## 화면이 로딩되지 않음

확인:

- GitHub Pages 배포가 성공했는지
- 브라우저 콘솔에 에러가 있는지
- `assets/` 경로가 올바르게 배포되었는지

조치:

```bash
npm ci
npm run build
```

## 카메라 또는 이동이 이상함

- 브라우저가 포인터 잠금을 허용했는지 확인
- 마우스 클릭 후 이동해 보기
- ESC로 일시정지 후 다시 시작

## 신호등이 보이지 않음

- 횡단보도 근처로 이동
- 저사양 모드 사용 시에도 신호 표시가 유지되는지 확인
- 배포본이 최신인지 확인

## P-03 시야 제한이 약함

- 설정에서 시야 효과 강도를 확인
- 체험 효과가 켜져 있는지 확인
- P-03 유형이 선택되어 있는지 확인
