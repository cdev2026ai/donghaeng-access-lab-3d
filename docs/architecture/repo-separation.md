# 학생용/개발자용 레포 분리

## 분리 이유

학생용 레포에는 체험 몰입을 방해하는 개발자 기능을 포함하지 않는다. 좌표 확인, 체크포인트 이동, 결과 강제 열기 같은 기능은 별도 개발자용 레포에서 관리한다.

## 레포 구분

| 레포 | 역할 | 공개 대상 |
|---|---|---|
| `donghaeng-access-lab-3d` | 학생용 체험 시뮬레이터 | 학생, 수업 참여자 |
| `donghaeng-access-lab-3d-dev` | 개발자용 검수 시뮬레이터 | 강사, 운영자, 개발자 |

## 운영 원칙

- 학생용은 안정성과 체험 흐름을 우선한다.
- 개발자용은 검수 편의성과 좌표 확인을 우선한다.
- 개발자용에서 검증된 변경만 학생용으로 반영한다.
