## 직접 확인 필요

현재 제작 환경에서는 실제 마우스 잠금 기반 전체 주행을 자동 검수하지 못했습니다. Chrome 또는 Edge에서 다음을 확인해야 합니다.

- 신호등 본체가 접근 방향에서 실제로 보이는지
- 숫자가 거울처럼 반전되지 않는지
- 종료 임박 점멸이 과도하지 않은지
- P-02가 실제 조작에서도 신호 종료 전에 도착하기 어려운지
- 횡단 중 신호 종료 후 차량이 움직이지 않는지
