## 직접 확인이 필요한 항목

현재 자동 검수 환경에서는 실제 사용자 마우스 잠금과 전체 경로 주행을 끝까지 수행하지 못했습니다. Chrome 또는 Edge에서 다음을 직접 확인해야 합니다.

1. P-03 중간 강도에서 이전 버전보다 중앙 가용 시야가 조금 더 좁아졌는지
2. O-04 점자블록 단절 구간에서 주변부 암화와 방향 확신도 저하가 함께 나타나는지
3. 낮음·끄기 선택 시 즉시 완화되는지
4. 중앙 시야와 흰지팡이 탐지가 유지되어 미션 진행이 가능한지
