## 경고

Three.js를 하나의 번들로 포함해 JavaScript 청크가 500KB를 초과한다는 Vite 경고가 있습니다. 현재 수업용 단일 장면에서는 기능 오류가 아니며, 후속 최적화 단계에서 코드 분할을 검토할 수 있습니다.

## 수동 검수가 필요한 항목

현재 제작 환경의 GPU·EGL 제한으로 실제 WebGL 화면 자동 캡처는 완료하지 못했습니다. Chrome 또는 Edge에서 다음을 확인해야 합니다.

1. 네 유형의 카메라 높이와 속도 차이
2. 휠체어 모드 O-01·O-03·O-05 통과 제한
3. 고령 보행 모드 0.3초 입력 지연
4. 시각장애 모드 F 키 지팡이 탐지
5. O-04 방향 편차와 결과 기록
6. GitHub Pages에서 새로고침 후 자산 정상 로드

상세 항목은 `V0.3_TEST_CHECKLIST.md`를 사용합니다.
