# v0.9.4 테스트 체크리스트

## 기본 실행

- [ ] `npm ci`가 정상 완료된다.
- [ ] `npm run dev`로 로컬 서버가 실행된다.
- [ ] 시작 화면에서 버전이 v0.9.4로 표시된다.

## 목적지·경로 변경 확인

- [ ] 버스정류장 목적지가 기존보다 왼쪽 뒤편으로 이동했다.
- [ ] 횡단보도 이후 청록색 바닥 경로 마커가 보인다.
- [ ] 경로 마커가 불법 주차 차량(O-03) 옆 좁은 통로를 지나 목적지 방향으로 이어진다.
- [ ] O-03 근처에 노란색 경유 지점 마커가 표시된다.
- [ ] 화면 왼쪽 진행 카드에 `불법 주차 구간` 항목이 표시된다.

## O-03 체험 영향 확인

- [ ] 횡단보도 이후 목적지로 이동할 때 O-03 구간이 현재 구간으로 잡힌다.
- [ ] O-03 구간에서 속도 저하 또는 통로 폭 제약이 체감된다.
- [ ] 휠체어 사용자(P-01)는 차량과 울타리 사이 통로 폭이 충분하지 않음을 확인할 수 있다.
- [ ] 시각장애인(P-03)은 흰지팡이 탐지로 O-03 구간의 차량 또는 울타리를 확인할 수 있다.

## 앱 도움 ON 안내 확인

- [ ] 앱 도움 ON에서 O-03을 아직 지나지 않은 경우 불법 주차 구간 경유 안내가 제공된다.
- [ ] 앱 안내에 경유 지점까지 거리와 방향이 표시된다.
- [ ] O-03 구간에 가까워지면 장애물명, 거리, 통과 조건 안내가 표시된다.

## 목적지 완료 판정

- [ ] O-03 구간을 관찰한 뒤 목적지에 도착하면 결과 화면이 열린다.
- [ ] O-03 구간을 관찰하지 않고 목적지로 접근하면 완료가 보류되고 경유 안내가 나온다.

## 회귀 테스트

- [ ] 보행 신호와 신호등 오브젝트 표시가 정상 작동한다.
- [ ] P-02 고령 보행자는 신호 시간 내 횡단이 어렵게 유지된다.
- [ ] P-03 시야 제한 효과가 유지된다.
- [ ] P-02 고령자 희뿌연 시야 효과가 유지된다.
- [ ] P-01 요철 진동 효과가 유지된다.
