## 기본 실행

- [ ] `npm ci`가 정상 완료된다.
- [ ] `npm run build`가 오류 없이 완료된다.
- [ ] 시작 화면 상단 버전 표기가 v0.9.5로 표시된다.

## 점자블록(O-04) 변경 확인

- [ ] 횡단보도 북측 신호등 이후부터 점자블록이 목적지 방향으로 이어진다.
- [ ] 점자블록이 이전보다 더 길게 연속 배치되어 있다.
- [ ] 점자블록 중간에 훼손·단절 구간이 시각적으로 표현된다.
- [ ] O-04 라벨이 점자블록 단절 구간 근처에 표시된다.

## 볼라드·입간판(O-05) 변경 확인

- [ ] 볼라드 3개가 버스정류장 앞 통로에 배치된다.
- [ ] 입간판이 목적지 앞 진입 구간에 놓여 있다.
- [ ] O-05 라벨이 목적지 직전 구간에 표시된다.
- [ ] 목적지 직전 구간에서 통로 폭 제약이 체감된다.

## 권장 경로·체험 흐름 확인

- [ ] 청록색 경로 마커가 불법 주차 차량 → 점자블록 → 볼라드 앞 통로 → 목적지 흐름으로 이어진다.
- [ ] O-03 경유 후 O-04와 O-05가 자연스럽게 같은 이동선 위에 놓인다.
- [ ] 목적지로 이동하는 과정에서 O-04와 O-05가 현재 구간 또는 주변 장벽으로 인지된다.

## 앱 도움 ON 안내 확인

- [ ] 앱 도움 ON에서 파손 점자블록에 대한 안내가 제공된다.
- [ ] 앱 도움 ON에서 목적지 앞 볼라드·입간판 통로 안내가 제공된다.
- [ ] 앱 안내 문구가 마지막 접근 구간의 거리·방향·통로 폭 맥락을 유지한다.
