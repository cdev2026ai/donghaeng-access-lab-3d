# 이동 보조 앱 OFF/ON 기능 명세 v0.9

## 교육 목표

학생이 앱 도움 전·후의 이동 차이를 체험하고, 이동 보조 앱이 단순히 장애물 이름을 분류하는 것을 넘어 어떤 맥락 정보를 제공해야 하는지 발견하도록 합니다.

## 앱 도움 없음

- 환경 자체의 표지, 보행 신호, 점자블록, 음향 정보만 사용합니다.
- 장애물 거리·방향·통과 가능 여부를 자동으로 안내하지 않습니다.
- 신호 잔여 시간과 예상 횡단 시간을 비교하지 않습니다.
- 시각장애인 조건에서 음향신호가 꺼져 있으면 정확한 신호 문구를 숨깁니다.
- 결과 화면에 부족했거나 확인하기 어려웠던 정보를 정리합니다.

## 앱 도움 있음

### 입력 Input

- 이동약자 유형
- 현재 위치와 진행 방향
- 현재 이동 속도
- 장애물 종류·거리·통과 폭
- 경사로 위치
- 보행 신호 상태와 잔여 시간
- 차량 정지·접근 상태

### 처리 Process

- 유형별 이동 조건 적용
- 예상 횡단 시간 계산
- 신호 잔여 시간과 비교
- 경사로·우회로 방향 계산
- 위험도와 안내 우선순위 결정

### 출력 Output

- 짧은 비모달 자막
- 선택적 TTS
- 신호 상태와 남은 시간
- 예상 횡단 시간과 시간 여유
- 출발·대기·주의 권고
- 경사로·우회로 방향과 거리
- 장애물까지 거리와 통과 조건
- 횡단 방향 각도와 남은 거리

## 안내 우선순위

1. 즉시 위험: 신호 위반, 차량 접근, 충돌 직전
2. 주의: 신호 종료 임박, 시간 부족, 통과 폭 부족
3. 일반: 경사로·우회로·방향·남은 거리

## 비모달 원칙

- 안내 패널은 이동을 강제로 중단하지 않습니다.
- 동일 안내를 지나치게 반복하지 않습니다.
- 사용자가 설정에서 TTS를 끌 수 있습니다.
- 저사양 모드에서도 텍스트 안내는 유지합니다.

## 결과 연결

- 앱 OFF: 부족했던 정보 목록을 강조
- 앱 ON: 실제 제공된 안내와 추가 개선점을 강조
- 같은 유형·같은 경로에서 앱 도움 상태를 바꿔 재체험할 수 있음
- 결과 저장·비교 기능은 넣지 않고 워크북 활동으로 연결
