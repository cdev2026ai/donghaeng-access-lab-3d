# v0.6 미션·결과·비교 화면 테스트 체크리스트

## A. 시작·미션 브리핑

- [ ] 시작 안내에서 이동약자 유형 선택 화면으로 이동한다.
- [ ] 유형 선택 후 바로 3D 체험이 시작되지 않고 미션 브리핑이 열린다.
- [ ] 미션 브리핑에 선택 유형, 핵심 병목, 기본 속도가 표시된다.
- [ ] 미션 브리핑에 학교 정문→버스정류장 경로와 4개 목표가 표시된다.
- [ ] `유형 다시 선택`이 정상 작동한다.
- [ ] `미션 시작` 후에만 포인터 잠금과 이동이 시작된다.
- [ ] 미션 시작 전 장애물 이벤트가 끼어들지 않는다.

## B. 실시간 미션 UI

- [ ] 장애물 이벤트 완료 수가 HUD에 반영된다.
- [ ] 안전 판단 횟수가 HUD에 반영된다.
- [ ] 목적지 도착 전/후 상태가 구분된다.
- [ ] 일시정지 화면에서 미션 안내를 다시 열 수 있다.

## C. 결과 화면

- [ ] 목적지 도착 후 결과 화면이 열린다.
- [ ] 경로·이벤트·안전 판단·재시도 상태가 표시된다.
- [ ] 이동 기록, 이벤트 기록, 상태 지표가 표시된다.
- [ ] 체험 해석 문구가 생성된다.
- [ ] 성찰 질문 3개를 작성하고 저장할 수 있다.
- [ ] 같은 유형 다시 체험, 다른 유형 체험, 시작 화면 이동이 작동한다.

## D. 저장·내보내기

- [ ] 완료 결과가 브라우저에 자동 저장된다.
- [ ] 성찰 내용을 저장한 후 비교 화면을 다시 열어도 유지된다.
- [ ] 개별 JSON 파일을 저장할 수 있다.
- [ ] 개별 CSV 파일을 저장할 수 있다.
- [ ] 한글 CSV가 UTF-8 BOM으로 정상 열린다.

## E. 비교 화면

- [ ] 상단 ⇄ 버튼과 결과 화면 버튼으로 비교 화면을 열 수 있다.
- [ ] 기록이 없을 때 빈 상태 안내가 표시된다.
- [ ] 기록 2~4개를 선택할 수 있다.
- [ ] 5번째 기록 선택은 제한된다.
- [ ] 비교 표에 유형별 지표가 나란히 표시된다.
- [ ] 비교 관찰 문구가 생성된다.
- [ ] 비교 CSV를 저장할 수 있다.
- [ ] 저장 기록 지우기가 작동한다.

## F. 회귀 테스트

- [ ] v0.5.2 시야 효과 강도와 흰지팡이가 유지된다.
- [ ] v0.4.1 시작 순서 보호가 유지된다.
- [ ] 장애물 5종과 이벤트 5단계가 유지된다.
- [ ] `npm run build`가 성공한다.
- [ ] Chrome/Edge에서 콘솔 오류 없이 실행된다.
