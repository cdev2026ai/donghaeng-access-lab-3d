# v0.9.2 테스트 체크리스트

## 시각장애인 시야 효과
- [ ] P-03 시각장애인·흰지팡이 사용자를 선택한다.
- [ ] 기본값이 `강함`으로 설정되어 있는지 확인한다.
- [ ] 체험 시작 직후 주변부가 어둡고 중앙 시야만 상대적으로 남는지 확인한다.
- [ ] 설정에서 `끄기`를 선택하면 시야 제한이 사라지는지 확인한다.
- [ ] `낮음`, `중간`, `강함`, `매우 강함` 차이가 분명한지 확인한다.
- [ ] O-04 파손 점자블록 구간에서 시야 제한과 방향 정보 부담이 더 강하게 느껴지는지 확인한다.

## 휠체어 요철 진동
- [ ] P-01 수동 휠체어 사용자를 선택한다.
- [ ] O-04 파손 점자블록·거친 노면 구간으로 이동한다.
- [ ] 화면 상하 진동이 분명히 발생하는지 확인한다.
- [ ] 상태 UI에 `요철 진동` 배지가 표시되는지 확인한다.
- [ ] 설정에서 `화면 흔들림`을 끄면 진동이 사라지는지 확인한다.

## 고령 보행자 음성 안내
- [ ] P-02 고령 보행자·지팡이 사용자를 선택한다.
- [ ] 앱 도움 ON으로 체험을 시작한다.
- [ ] 음성 안내가 P-00 또는 P-03보다 작게 들리는지 확인한다.
- [ ] 자막 안내는 정상적으로 유지되는지 확인한다.
- [ ] 음향신호기 ON 상태에서 신호음도 작게 들리는지 확인한다.

## 회귀 검수
- [ ] 신호등 본체의 사람 아이콘과 숫자가 계속 표시된다.

- [ ] 앱 도움 OFF/ON 조건 선택 흐름이 유지된다.
- [ ] 결과 화면에서 관찰형 기록이 정상 표시된다.
