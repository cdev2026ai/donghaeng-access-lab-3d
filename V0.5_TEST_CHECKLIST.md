# v0.5.2 체험 효과와 상태 UI 테스트 체크리스트

## A. 시작·기본 흐름

- [ ] 시작 안내 → 이동약자 유형 선택 → 체험 시작 순서가 유지된다.
- [ ] 유형 선택 전 장애물 이벤트가 열리지 않는다.
- [ ] 시작 후 첫 이동을 해야 O-01 이벤트가 시작된다.

## B. 상태 UI

- [ ] 우측에 피로도·불안감·방향 확신도·시간 압박이 표시된다.
- [ ] 이동하면 피로도가 증가하고 멈추면 조금 회복된다.
- [ ] 장애물 접근 시 불안감이 소폭 증가한다.
- [ ] 안전 판단을 선택하면 불안감이 감소하고 방향 확신도가 회복된다.
- [ ] 위험 판단을 선택하면 불안감이 증가한다.
- [ ] 충돌하면 불안감이 증가하고 방향 확신도가 감소한다.
- [ ] 현재 구간에 따라 적용 효과 배지가 바뀐다.

## C. 유형별 효과

- [ ] P-00은 상태 변화가 가장 완만하다.
- [ ] P-01은 경사·거친 노면에서 피로 증가와 감속이 나타난다.
- [ ] P-02는 이동 중 피로도가 가장 빠르게 누적된다.
- [ ] P-03은 시작부터 방향 확신도가 다른 유형보다 낮다.
- [ ] P-03에서 O-04 진입 시 방향 확신도와 정보 명료도가 감소한다.

## D. 시야·화면 효과

- [ ] P-03에서 주변부 시야 제한이 나타난다.
- [ ] 설정에서 시야 효과를 끄면 주변부 효과가 사라진다.
- [ ] 낮음과 중간 강도의 차이가 확인된다.
- [ ] 피로가 높아지면 하단 어두운 효과가 증가한다.
- [ ] 불안감이 높아지면 화면 가장자리 맥동 효과가 나타난다.
- [ ] 화면 흔들림 OFF 시 거친 노면과 불안 흔들림이 중단된다.
- [ ] 저사양 모드에서도 화면이 깨지지 않는다.
- [ ] 고대비 UI를 켜면 패널과 안내 문구 대비가 강화된다.

## E. 흰지팡이

- [ ] P-03에서 화면 오른쪽 아래 흰지팡이가 보인다.
- [ ] 다른 유형에서는 흰지팡이가 보이지 않는다.
- [ ] F 키를 누르면 흰지팡이가 좌우로 움직인다.
- [ ] 장애물 감지 시 안내 문구와 탐지 횟수가 갱신된다.
- [ ] 설정에서 흰지팡이 표시를 끌 수 있다.

## F. 횡단보도 시간 압박

- [ ] 횡단보도 구간에서 보행 신호 잔여 시간이 줄면 시간 압박이 증가한다.
- [ ] 횡단보도를 벗어나면 시간 압박이 감소한다.
- [ ] 최대 시간 압박이 결과 화면에 기록된다.

## G. 결과 화면

- [ ] 최종 피로도가 표시된다.
- [ ] 최대 불안감이 표시된다.
- [ ] 최저 방향 확신도가 표시된다.
- [ ] 최대 시간 압박이 표시된다.
- [ ] 기존 이벤트 로그와 충돌·차단 기록이 유지된다.

## H. 회귀 테스트

- [ ] 장애물 5종이 모두 표시되고 통과 판정이 작동한다.
- [ ] 네 유형의 카메라 높이와 속도 차이가 유지된다.
- [ ] 이벤트 5단계가 정상 진행된다.
- [ ] R 키 초기화 후 상태 값도 유형 초기값으로 돌아간다.
- [ ] 유형 변경 시 이전 상태 기록이 남지 않는다.
- [ ] 목적지 도착 후 결과 화면이 열린다.


## 시야 효과 강화 회귀 테스트

- [ ] P-03 선택 직후 기본값 `중간`에서 주변부 제한이 즉시 눈에 띈다.
- [ ] 중앙 가용 시야는 남아 있고 화면 전체가 검게 가려지지 않는다.
- [ ] `낮음 → 중간 → 강함` 순서로 주변부 암화와 장면 명료도 변화가 분명히 증가한다.
- [ ] `끄기` 선택 시 시야 마스크와 장면 필터가 모두 제거된다.
- [ ] O-04 점자블록 단절 구간에서 기본 구간보다 시야·방향 정보 부담이 더 크게 표현된다.
- [ ] 저사양 모드에서도 주변부 마스크가 보이고, 과도한 블러만 제거된다.


## v0.5.2 추가 확인

- [ ] P-03 중간 강도에서 중앙 가용 시야가 v0.5.1보다 조금 더 좁게 보이는가
- [ ] 주변부 암화가 화면 가장자리뿐 아니라 중앙 주변부터 분명하게 시작되는가
- [ ] O-04 진입 시 시야 부담과 방향 확신도 저하가 동시에 뚜렷한가
- [ ] 시야 효과를 낮음 또는 끄기로 바꾸면 즉시 완화되는가
- [ ] 중앙 시야와 흰지팡이 탐지가 유지되어 체험 진행이 가능한가
