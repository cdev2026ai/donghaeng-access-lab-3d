# v0.9.4 불법 주차 차량 경유 경로 보완

## 변경 목적

이전 버전에서는 목적지인 버스정류장이 기존 이동선의 오른쪽에 가까워, 사용자가 횡단보도를 지난 뒤 불법 주차 차량(O-03)을 충분히 경험하지 않고도 목적지에 도착할 수 있었다. v0.9.4에서는 목적지를 더 멀리, 왼쪽 뒤편으로 이동하여 불법 주차 차량이 보도를 막은 좁은 통로를 자연스럽게 경유하도록 수정했다.

## 반영 사항

### 1. 목적지 위치 조정

- 기존 목적지: 버스정류장 `x 4.7, z -22.5`
- 변경 목적지: 버스정류장 `x -5.8, z -26.0`
- 횡단보도 반대편에서 목적지로 이동하려면 O-03 구간 방향으로 이동해야 한다.

### 2. 권장 경로 표시 추가

횡단보도 이후부터 버스정류장까지 바닥에 청록색 경로 마커를 추가했다.

경로 흐름:

```text
횡단보도 이탈
→ 불법 주차 차량 옆 좁은 통로
→ 북서쪽 보도 이동
→ 버스정류장 도착
```

O-03 근처에는 노란색 경유 지점 마커와 안내 라벨을 추가했다.

### 3. 앱 안내 문구 보완

앱 도움 ON 상태에서 O-03을 아직 관찰하지 않은 경우 다음 안내가 제공된다.

- 불법 주차 차량 구간을 지나야 함
- 경유 지점까지 거리
- 진행 방향
- 차량과 울타리 사이 통로 폭 확인

### 4. 진행 체크 항목 추가

화면 왼쪽 진행 카드에 `불법 주차 구간` 체크 항목을 추가했다.

```text
보도 진입 → 횡단 → 불법 주차 구간 → 목적지
```

### 5. 목적지 도착 판정 보완

목적지에 가까이 가더라도 O-03 구간을 관찰하지 않았다면 바로 완료되지 않는다. 이 경우 다음 안내가 표시된다.

> 불법 주차 차량이 보도를 막은 좁은 통로를 관찰한 뒤 목적지에 도착해야 합니다.

## 교육적 의미

이 변경은 불법 주차 차량을 단순 배경 오브젝트가 아니라 이동 경로에 영향을 주는 환경 장벽으로 만든다. 학생들은 같은 목적지 이동 과정에서 다음 정보를 관찰하게 된다.

- 통로 폭이 충분한가?
- 휠체어 사용자는 통과 가능한가?
- 시각장애인은 차량과 울타리 경계를 어떻게 파악하는가?
- 이동 보조 앱은 장애물 이름뿐 아니라 거리, 방향, 통로 폭, 우회 가능성을 어떻게 안내해야 하는가?

## 테스트 포인트

1. 횡단보도 이후 목적지가 기존보다 왼쪽 뒤편에 있는지 확인한다.
2. 바닥의 청록색 경로 마커가 O-03 구간을 지나 목적지로 이어지는지 확인한다.
3. O-03 구간에 들어가면 현재 구간 안내가 불법 주차 차량으로 바뀌는지 확인한다.
4. 앱 도움 ON에서 O-03 경유 안내가 제공되는지 확인한다.
5. O-03을 관찰하지 않고 목적지로 가면 완료가 보류되는지 확인한다.
