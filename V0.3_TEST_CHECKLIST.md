# v0.3 이동약자 유형 시스템 테스트 체크리스트

## 공통

- [ ] 첫 화면에서 이동약자 유형 선택 화면이 열린다.
- [ ] P-00~P-03 카드 선택 상태가 시각적으로 구분된다.
- [ ] 선택한 유형 이름·핵심 병목·카메라 높이·기본 속도가 HUD에 표시된다.
- [ ] ESC → 이동약자 유형 바꾸기로 유형을 변경할 수 있다.
- [ ] 유형 변경 시 위치·시간·충돌·장애물 진행 상태가 초기화된다.
- [ ] R 키로 현재 유형의 시작 상태가 복구된다.

## P-00 비교 기준

- [ ] 카메라 높이가 약 1.65m이다.
- [ ] 기본 속도가 100%이다.
- [ ] O-01 턱을 정면으로 통과할 수 있다.
- [ ] O-03·O-05 좁은 통로를 통과할 수 있다.

## P-01 수동 휠체어 사용자

- [ ] 카메라 높이가 약 0.95m이다.
- [ ] 기본 속도가 70%이고 회전·좌우 이동이 느리다.
- [ ] O-01 턱 정면 통과가 차단된다.
- [ ] O-01 오른쪽 경사 통로로 우회할 수 있다.
- [ ] O-03 0.75m 통로에서 폭 부족으로 차단된다.
- [ ] O-05 좁은 볼라드 간격에서 정면 통과가 차단된다.

## P-02 고령 보행자

- [ ] 카메라 높이가 약 1.45m이다.
- [ ] 기본 속도가 55%이다.
- [ ] 이동 입력을 새로 시작할 때 약 0.3초 지연이 느껴진다.
- [ ] O-01에서 감속하지만 정면 통과가 가능하다.
- [ ] O-02 경사에서 가장 낮은 속도 배율이 적용된다.
- [ ] O-03·O-05는 느리지만 통과할 수 있다.

## P-03 시각장애인

- [ ] 카메라 높이가 약 1.55m이다.
- [ ] 기본 속도가 60%이다.
- [ ] 화면 하단에 F 키 지팡이 탐지 안내가 표시된다.
- [ ] F 키 사용 시 1.2m 안의 장애물 또는 감지 실패 메시지가 표시된다.
- [ ] O-01 턱에서 정면 통과가 차단된다.
- [ ] O-04 단절 구간에서 경로가 조금씩 옆으로 편차된다.
- [ ] 결과 화면에 지팡이 탐지 횟수와 방향 편차가 표시된다.
