# 체험 효과와 상태 UI 설계 — v0.5.2

## 1. 설계 원칙

1. 장애 자체를 단순화하거나 과장하지 않고 환경과 이동 조건의 상호작용을 표현한다.
2. 시각장애인 체험을 완전한 검은 화면으로 만들지 않는다.
3. 흔들림·블러·주변부 제한은 사용자가 끄거나 강도를 조절할 수 있어야 한다.
4. 상태 변화는 실제 행동과 장애물 이벤트에 연결한다.
5. 상태 수치는 교육적 비교를 위한 시뮬레이션 값이며 의학적 측정값이 아니다.

## 2. 상태 변수

### 피로도

- 범위: 0~100
- 이동 중 점진적으로 증가
- P-02 고령 보행자에서 가장 빠르게 증가
- P-01 휠체어 사용자도 경사 구간에서 증가율이 커짐
- O-02 경사, O-04 거친 노면, 충돌 시 추가 증가
- 멈추면 서서히 회복
- 높은 피로도에서는 최대 이동 속도가 최대 약 28% 감소

### 불안감

- 범위: 0~100
- 장애물 이벤트 접근 시 소폭 증가
- 위험 판단 선택 시 크게 증가
- 충돌·이동 차단 시 증가
- 안전 판단, 상황 재확인, 지팡이 탐지 시 감소
- 65 이상에서는 화면 가장자리에 약한 맥동 효과와 선택적 미세 흔들림 적용

### 방향 확신도

- 범위: 0~100
- 값이 높을수록 방향 정보가 안정적임
- P-03 시각장애인 모드는 낮은 초기값에서 시작
- O-04 점자블록 단절 구간에서 크게 감소
- F 키 지팡이 탐지, 안전 판단, 재확인으로 회복
- 충돌과 위험 판단 시 감소

### 시간 압박

- 범위: 0~100
- 횡단보도 구간에서 보행 신호 잔여 시간이 줄어들수록 증가
- 신호 대기 또는 횡단보도 이탈 시 감소
- 최종 결과에 최대 시간 압박 기록

### 정보 명료도

- 내부 계산 변수
- P-03 시각장애인 모드의 주변부 시야와 정보 접근 효과 강도에 사용
- O-04 점자블록 단절, O-03 차량 근접, O-05 시설물 밀집 구간에서 낮아짐
- 화면에는 별도 점수 대신 적용 효과 배지와 방향 확신도로 표현

## 3. 시각 효과

- 주변부 시야 제한: P-03에만 적용
- 낮음/중간/끄기 설정 제공
- 정보 명료도가 낮을수록 주변부가 어두워지고 약한 블러 적용
- 피로 누적 시 화면 하단에 약한 어두운 헤이즈 적용
- 불안 증가 시 화면 가장자리 붉은 맥동 적용
- 저사양 모드에서는 backdrop blur를 제거하고 단순 그라디언트로 대체
- 고대비 UI는 패널 테두리와 안내 문구 대비를 강화

## 4. 흰지팡이 시각화

- P-03 선택 시 1인칭 카메라 오른쪽 아래에 흰지팡이 표시
- 흰색 본체, 어두운 손잡이, 붉은 끝부분으로 구성
- F 키 또는 이벤트 상황인지 버튼 사용 시 좌우 탐색 애니메이션
- 장애물 감지 여부에 따라 탐색 각도와 하강 폭을 다르게 표현
- 설정에서 표시를 끌 수 있음

## 5. HUD 등급

- 0~24: 안정
- 25~49: 주의
- 50~74: 부담 증가
- 75~100: 부담 높음
- 방향 확신도는 역방향 기준으로 판정
- 전체 상태 등급은 네 상태 중 가장 부담이 큰 값을 기준으로 표시

## 6. 접근성 설정

- 체험 효과 전체 ON/OFF
- 시야 효과: 끄기 / 낮음 / 중간
- 화면 흔들림 ON/OFF
- 고대비 UI ON/OFF
- 흰지팡이 표시 ON/OFF
- 저사양 모드

## 7. 결과 기록

최종 화면에 다음 상태를 표시한다.

- 최종 피로도
- 최대 불안감
- 최저 방향 확신도
- 최대 시간 압박

이 값은 장애물별 이벤트 기록과 함께 체험 유형 간 비교와 성찰에 사용한다.


## v0.5.1 시야 효과 강화 규칙

- 기본 강도: 중간
- 낮음: 주변부 제한을 인지할 수 있지만 조작 학습에 방해되지 않는 수준
- 중간: 중앙 가용 시야가 분명히 좁아지고 주변부가 어두워지는 기본 체험 수준
- 강함: 비교·토론용 강화 수준. 화면을 완전히 가리지 않으며 설정에서 즉시 낮추거나 끌 수 있음
- O-03 불법 주차 차량, O-04 점자블록 단절, O-05 시설물 밀집 구간에서는 정보 명료도가 추가로 낮아짐
- 이 효과는 모든 시각장애 경험을 재현한다고 주장하지 않으며, 환경 정보 접근이 제한될 때 이동 판단이 어려워지는 구조를 교육적으로 표현함


## v0.5.2 시야 효과 추가 강화

- 기본 강도는 계속 `중간`으로 유지하되, 중앙 가용 시야를 이전보다 조금 더 좁힘
- 주변부 암화 시작 지점을 중앙에 더 가깝게 조정하여 터널 시야를 분명하게 표현
- 기본 정보 명료도와 O-03·O-04·O-05 구간의 정보 명료도를 추가로 낮춤
- 중간 강도에서도 채도·밝기 저하와 블러가 명확히 느껴지도록 보정
- O-04에서는 방향 정보 단절과 시야 부담이 함께 커지도록 설정
- 강함 옵션은 수업 비교 시연용이며, 불편하면 설정에서 낮음 또는 끄기로 즉시 변경 가능
