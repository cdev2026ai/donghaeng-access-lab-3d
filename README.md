# 동행 Access Lab 3D — 장애물 5종 구현

고등 7~12차시에서 사용할 이동약자 3D 체험 시뮬레이터의 **5단계 장애물 구현 버전(v0.2)**입니다.

## 이번 버전에서 추가된 기능

- O-01 높은 단차·보도 턱
  - 정면 통과 차단
  - 오른쪽 경사 통로 우회
- O-02 가파른 경사로
  - 경사 높이 변화
  - 이동 속도 55% 감소
- O-03 불법 주차 차량·좁은 통로
  - 차량·펜스 충돌
  - 좁은 통로 감속
  - 차도 우회 위험 구역 표시
- O-04 파손·끊긴 점자블록
  - 끊긴 유도선과 잘못 연결된 방향
  - 거친 노면 감속·미세 화면 흔들림
- O-05 볼라드·입간판
  - 좁은 간격과 접근 각도
  - 충돌 후 후진 재진입

## 공통 기능

- 장애물 접근·상황인지·판단·수행·결과 문구
- 장애물별 진행 상태와 통과 수 표시
- 충돌·이동 차단·거친 노면 체류 시간 기록
- 설정에서 장애물 5종 개별 ON/OFF
- 화면 흔들림 ON/OFF
- 기존 기본 장면·WASD 이동·신호·목적지·저사양 모드 유지
- GitHub Pages 자동 배포

## 아직 포함하지 않은 기능

다음 기능은 후속 버전에서 추가합니다.

- 이동약자 페르소나 3종과 비교 기준 모드
- 이동약자 × 장애물 15개 조합 규칙
- 페르소나별 카메라 높이·사용자 폭·최대 단차·경사 한계
- 피로도·불안감·방향 확신도
- 판단 선택 UI와 장애물별 결과 비교
- 개발자 모드와 동료 평가

## 실행

Node.js 22.12 이상을 권장합니다.

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```

## 조작

- `W A S D` 또는 방향키: 이동
- 마우스: 시점 회전
- `ESC`: 일시정지 및 마우스 해제
- `R`: 학교 정문으로 재설정

## 장애물 권장 통과 순서

1. O-01 단차 오른쪽 경사 통로
2. O-02 가파른 경사 플랫폼
3. 횡단보도
4. O-03 차량 오른쪽 좁은 통로
5. O-04 파손 점자블록
6. O-05 볼라드 사이 통로
7. 버스정류장 도착 구역

## 교육적 고지

이 시뮬레이터는 장애인의 실제 경험 전체를 재현하지 않습니다. 현재 버전은 이동 장벽의 물리적 구조와 기본 반응을 검증하기 위한 기준 보행 모드이며, 이동약자별 차이는 다음 단계에서 적용합니다.
