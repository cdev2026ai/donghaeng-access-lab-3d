# 동행 Access Lab 3D — 시야 체험 효과 강화 v0.5.2

고등 7~12차시에서 활용하는 이동약자 3D 체험 시뮬레이터입니다. 기본 3D 환경, 장애물 5종, 이동약자 유형 시스템, 상황인지–판단–동작수행 이벤트 위에 체험 효과와 상태 UI를 적용했습니다.

## v0.5.2 보완 범위

- 피로도·불안감·방향 확신도·시간 압박 실시간 상태 UI
- 이동 속도, 경사, 거친 노면, 충돌, 판단 선택에 따른 상태 변화
- 고령 보행자의 피로 누적과 반응 지연
- 휠체어 사용자의 경사·단차·좁은 통로 수행 부담
- 시각장애인 모드의 주변부 시야 제한을 v0.5.1보다 조금 더 강화
- 중앙 가용 시야 추가 축소, 주변부 암화 경계 강화, 채도·밝기·거리 명료도 추가 저하
- 시야 효과 강도 `끄기 / 낮음 / 중간 / 강함` 제공, 기본값 `중간`
- 점자블록 단절 구간의 방향 확신도 하락
- 횡단보도 보행 신호 잔여 시간에 따른 시간 압박
- 불안·피로 상태에 따른 약한 화면 효과와 이동 성능 변화
- 시각장애인 모드의 1인칭 흰지팡이 3D 모델과 F 키 탐지 애니메이션
- 체험 효과 ON/OFF, 시야 효과 강도, 고대비 UI, 흰지팡이 표시 설정
- 최종 결과 화면의 상태 요약

## 이전 단계 유지

- 학교 정문 → 보도 → 횡단보도 → 버스정류장 3D 환경
- 장애물 O-01~O-05
- P-00 비교 기준, P-01 수동 휠체어, P-02 고령 보행, P-03 시각장애인 유형
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 장애물별 판단 선택지, 충돌·차단·판단 시간·수행 시간 기록

## 상태 규칙 요약

| 상태 | 증가 요인 | 완화 요인 |
|---|---|---|
| 피로도 | 이동 지속, 경사, 거친 노면, 충돌 | 멈춤과 휴식 |
| 불안감 | 장애물 접근, 위험 판단, 충돌, 반복 차단 | 안전 판단, 상황 재확인, 지팡이 탐지 |
| 방향 확신도 | 연속된 유도 정보, 상황 확인, 안전 판단 | 점자블록 단절, 충돌, 위험 판단 |
| 시간 압박 | 횡단보도 보행 신호 잔여 시간 감소 | 횡단보도 이탈, 신호 대기 |

## 주요 조작

- WASD / 방향키: 이동
- 마우스: 시점 회전
- Enter / E: 이벤트 다음 단계
- 1 / 2 / 3: 판단 선택
- F: 시각장애 유형의 흰지팡이 탐지
- ESC: 일시정지
- R: 현재 유형으로 처음부터

## 문서

- `EXPERIENCE_EFFECTS.md`: 체험 효과·상태 계산 규칙
- `EVENT_SYSTEM.md`: 이벤트 상태·판정 규칙
- `PERSONAS.md`: 이동약자 유형 규칙
- `OBSTACLES.md`: 장애물 규칙
- `V0.5_TEST_CHECKLIST.md`: 기능 검수 목록
- `BUILD_REPORT.md`: 빌드·정적 검수 결과

## 아직 포함하지 않은 기능

- 사용자 계정과 서버 데이터베이스
- 유형별 결과 CSV/JSON 내보내기
- 교사 대시보드와 학급 통계
- 실시간 다중 사용자
- 동료 평가 입력 화면

## 교육적 고지

이 시뮬레이터는 장애인의 실제 경험 전체를 재현하지 않습니다. 특정 이동 조건과 환경 장벽의 상호작용을 구조적으로 관찰하기 위한 교육용 모델입니다. 시각장애인 효과도 완전한 암전이 아니라 정보 접근의 제한과 유도 정보 단절을 중심으로 표현합니다.
