# 동행 Access Lab 3D — 기본 환경 MVP

고등 7~12차시에서 사용할 이동약자 3D 체험 시뮬레이터의 **4단계 기본 3D MVP**입니다.

## 현재 구현 범위

- 학교 정문 → 보도 → 횡단보도 → 버스정류장 기본 장면
- Scene, Camera, Light, Object, Control 구성
- 1인칭 WASD/방향키 이동과 마우스 시점 회전
- 기본 건물, 가로수, 가로등, 교통신호, 목적지 표시
- 미션 HUD, 현재 구간, 거리, 속도, 이동 거리, 신호 상태
- 시작·일시정지·설정·완료 화면
- 출발 위치 재설정
- 저사양 모드와 상황 안내 ON/OFF
- GitHub Pages 배포 워크플로

## 아직 포함하지 않은 기능

다음 기능은 후속 버전에서 추가합니다.

- 이동약자 페르소나 3종 적용
- 장애물 5종과 15개 조합 규칙
- 단차·폭·경사별 충돌 판정
- 피로도·불안감·방향 확신도
- 장애물 접근→인지→판단→수행→결과 이벤트
- 비교 모드·개발자 모드·교차 평가

## 실행

Node.js 22.12 이상을 권장합니다.

```bash
npm install
npm run dev
```

브라우저에서 표시된 로컬 주소를 엽니다.

## 빌드

```bash
npm run build
npm run preview
```

빌드 결과는 `dist/`에 생성됩니다.

## 조작

- `W A S D` 또는 방향키: 이동
- 마우스: 시점 회전
- `ESC`: 일시정지 및 마우스 해제
- `R`: 학교 정문으로 재설정

## 프로젝트 구조

```text
src/
├─ main.ts        # 3D 장면, 이동, HUD, 화면 상태
└─ style.css      # 전체 UI 스타일
.github/workflows/
└─ deploy.yml     # GitHub Pages 자동 배포
```

## GitHub Pages

1. 저장소에 프로젝트를 업로드합니다.
2. GitHub 저장소의 `Settings → Pages`에서 Source를 `GitHub Actions`로 선택합니다.
3. `main` 브랜치에 push하면 자동으로 빌드·배포됩니다.

`vite.config.ts`는 상대 경로 배포를 위해 `base: './'`로 설정되어 있습니다.

## 교육적 고지

이 MVP는 기본 공간과 조작을 검증하기 위한 버전입니다. 장애인의 실제 경험 전체를 재현하지 않으며, 후속 단계에서도 특정 이동 조건과 환경 장벽의 상호작용을 설명하는 교육용 모델로 사용합니다.
