# 동행 Access Lab 3D — 미션·결과 v0.6.2

고등 7~12차시에서 활용하는 이동약자 3D 체험 시뮬레이터입니다. v0.5.2의 장애물·페르소나·상황인지–판단–수행 이벤트·체험 효과 위에 **미션 브리핑과 학습형 결과 화면**을 적용했습니다.

성찰 작성, 교차 체험 비교, 결과 저장·내보내기는 워크북과 팀 활동에서 이미 수행하므로 시뮬레이터에서는 제외했습니다. 화면 기능을 줄여 수업 흐름이 중복되지 않도록 구성했습니다.

## v0.6.2 포함 기능

- 이동약자 유형 선택 후 미션 브리핑
- 학교 정문 → 보도 → 횡단보도 → 버스정류장 경로 안내
- 활성 장애물·상황인지·판단·동작수행 목표
- 실시간 미션 체크리스트
- 경로·이벤트·안전 판단·재시도 달성 상태
- 이동 기록·상태 지표·장애물별 이벤트 결과
- 체험 기록 기반 자동 해석 문구
- 같은 유형 다시 체험
- 다른 이동약자 유형으로 다시 체험
- 결과 화면 하단 버튼을 1열 대표 행동 + 2열 보조 행동 구조로 정리

## 제외한 중복 기능

- 시뮬레이터 안의 성찰 문항 작성
- 완료 기록의 브라우저 저장
- 교차 체험 비교 화면
- JSON·CSV 내보내기

성찰과 교차 체험 비교는 워크북·팀 토의·발표 활동에서 진행합니다.

## 이전 단계 유지

- 학교 정문 → 버스정류장 기본 3D 환경
- 장애물 O-01~O-05
- P-00 비교 기준, P-01 휠체어, P-02 고령 보행, P-03 시각장애인
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 피로도·불안감·방향 확신도·시간 압박 상태 UI
- 강화된 시각장애인 시야 효과와 보이는 흰지팡이

## 주요 조작

- WASD / 방향키: 이동
- 마우스: 시점 회전
- Enter / E: 이벤트 다음 단계
- 1 / 2 / 3: 판단 선택
- F: 시각장애 유형의 흰지팡이 탐지
- ESC: 일시정지
- R: 현재 유형으로 처음부터

## 교육적 고지

이 시뮬레이터는 장애인의 실제 경험 전체를 재현하지 않습니다. 결과 화면은 사람의 능력을 평가하는 점수가 아니라, 이동 조건과 환경 장벽이 어떻게 상호작용했는지 확인하기 위한 교육 자료입니다.
