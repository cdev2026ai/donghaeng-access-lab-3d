# 동행 Access Lab 3D · v0.9.6

v0.9.5에서 의도했던 **점자블록과 볼라드·입간판 목적지 앞 재배치**가 실제 개발 소스에 확실히 반영되도록 수정한 버전입니다.

## 수정 배경

이전 덮어쓰기본에는 루트 `index.html`과 문서는 포함되어 있었지만, 실제 3D 배치 코드가 들어 있는 `src/main.ts`가 포함되지 않았습니다. 그래서 덮어쓰기본만 적용하면 화면에서 볼라드, 입간판, 훼손된 점자블록 위치가 바뀌지 않을 수 있었습니다.

## 핵심 변경

- 덮어쓰기본에 `src/main.ts` 포함
- 횡단보도 신호등 이후부터 점자블록이 목적지 방향으로 길게 이어짐
- 점자블록 중간 일부 훼손·단절 구간 표시
- 볼라드·입간판을 버스정류장 앞 통로에 배치
- 권장 경로가 `O-03 → O-04 → O-05 → 목적지` 흐름으로 이어짐

## 실행

```bash
npm ci
npm run dev
```

## 확인 순서

1. 횡단보도를 건넌다.
2. 불법 주차 차량 옆 좁은 통로를 지난다.
3. 신호등 이후부터 목적지 방향으로 이어지는 점자블록을 확인한다.
4. 점자블록 중간 훼손 구간을 확인한다.
5. 버스정류장 앞 볼라드·입간판을 확인한다.
