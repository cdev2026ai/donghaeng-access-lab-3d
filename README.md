# 동행 Access Lab 3D — 미션·결과·비교 v0.6

고등 7~12차시에서 활용하는 이동약자 3D 체험 시뮬레이터입니다. v0.5.2의 장애물·페르소나·상황인지–판단–수행 이벤트·체험 효과 위에 **미션 브리핑, 학습형 결과, 성찰, 교차 비교 화면**을 적용했습니다.

## v0.6 추가 기능

- 유형 선택 후 미션 브리핑 화면
- 학교 정문 → 보도 → 횡단보도 → 버스정류장 경로 안내
- 활성 장애물·상황인지·판단·동작수행·결과 목표
- 실시간 미션 체크리스트
- 경로·이벤트·안전 판단·재시도 달성 상태
- 체험 기록 기반 자동 해석 문구
- 성찰 질문 3종과 브라우저 저장
- 완료 기록 최대 8회 저장
- 2~4개 결과 교차 비교
- 개별 JSON/CSV 및 비교 CSV 내보내기
- 저장 기록 삭제

## 이전 단계 유지

- 학교 정문 → 버스정류장 기본 3D 환경
- 장애물 O-01~O-05
- P-00 비교 기준, P-01 휠체어, P-02 고령 보행, P-03 시각장애인
- 접근 → 상황인지 → 판단 → 동작수행 → 결과 이벤트
- 피로도·불안감·방향 확신도·시간 압박 상태 UI
- 강화된 시각장애인 시야 효과와 보이는 흰지팡이

## 주요 조작

- WASD / 방향키: 이동
- 마우스: 시점 회전
- Enter / E: 이벤트 다음 단계
- 1 / 2 / 3: 판단 선택
- F: 시각장애 유형의 흰지팡이 탐지
- ESC: 일시정지
- R: 현재 유형으로 처음부터
- 상단 ⇄: 저장된 결과 비교

## 데이터 저장

- 완료 결과와 성찰은 현재 브라우저의 localStorage에만 저장
- 최대 최근 8회 기록
- 계정·서버·개인정보 수집 없음
- JSON/CSV 파일로 직접 내보내기 가능

## 교육적 고지

이 시뮬레이터는 장애인의 실제 경험 전체를 재현하지 않습니다. 결과 비교는 사람의 능력이나 이동 시간을 평가하는 순위가 아니라, 이동 조건과 환경 장벽의 상호작용을 관찰하기 위한 교육 자료입니다.
