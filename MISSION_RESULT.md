# 체험 안내·관찰 결과 화면 v0.9

## 명칭 변경

- `미션 브리핑` → `체험 안내`
- `미션 성공` → `체험 경로를 완료했습니다`
- `안전 판단 점수` → 신호·횡단·이동 관찰 기록

## 체험 안내

- 선택 유형과 핵심 병목
- 앱 도움 없음/있음 조건
- 학교 정문 → 버스정류장 경로
- 횡단보도 신호와 앱 정보 관찰 포인트
- 조작법과 교육적 고지

## 관찰 결과

### 신호·횡단

- 신호 대기 시간
- 녹색 시작 후 출발 지연
- 출발 시 남은 신호 시간
- 횡단 소요 시간
- 신호 종료 전 완료 여부
- 경사로·방향 정렬
- 위험 진입·차량 비상 정지

### 이동

- 소요 시간과 이동 거리
- 관찰한 장애물
- 멈춤과 경로 재조정
- 이동 차단·충돌
- 지팡이 탐지와 방향 편차

### 앱 요구사항

- 부족했거나 확인하기 어려웠던 정보
- 이동 보조 앱이 제공해야 할 정보
- 앱 ON 체험에서 실제 제공된 안내와 추가 개선점

## 다음 행동

- 앱 도움 상태를 바꾸어 같은 조건 재체험
- 다른 이동약자 유형 체험
- 같은 조건 다시 체험
- 시작 화면

시뮬레이터 안에서 성찰을 입력하거나 결과를 저장하지 않습니다. 워크북과 팀 활동에서 앱 기능 요구사항으로 이어갑니다.
