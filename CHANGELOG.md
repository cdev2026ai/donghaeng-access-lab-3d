# 변경 기록

## v0.2.0 — obstacles

- 높은 단차·보도 턱과 우회 경사 통로 추가
- 가파른 경사 플랫폼과 속도 저하 구간 추가
- 불법 주차 차량·펜스·좁은 통로 추가
- 파손·끊긴 점자블록과 거친 노면 효과 추가
- 볼라드·입간판과 재진입 경로 추가
- 장애물 접근→인지→판단→수행→결과 안내 추가
- 장애물 진행 패널과 충돌·차단·거친 노면 결과 기록 추가
- 장애물 개별 ON/OFF와 화면 흔들림 설정 추가
