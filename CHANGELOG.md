# CHANGELOG

## v0.9.3 - 고령 보행자 시야 효과 보완

### Added
- P-02 고령 보행자 전용 약한 희뿌연 시야 효과 추가
- 고령 보행자 모드에서 약한 흐림, 낮은 대비, 채도 저하, 밝은 안개 오버레이 적용
- 피로도와 시간 압박이 높아질수록 고령자 시야 흐림과 대비 저하가 소폭 증가
- 상태 UI의 적용 효과 목록에 `약한 흐림·대비 저하` 표시
- 고대비 UI 사용 시 고령자 시야 효과가 과도하게 흐려지지 않도록 별도 필터 적용

### Changed
- P-02 초기 visionClarity 값을 96에서 86으로 조정
- 설정 화면 설명을 시각장애인 시야 제한과 고령자 약한 흐림 효과로 분리해 이해하기 쉽게 수정
- 시작 화면 버전 표기를 v0.9.3으로 갱신

### 유지
- P-03 시각장애인 주변부 시야 제한, 흰지팡이, 방향 정보 부족 효과 유지
- P-01 휠체어 사용자의 요철 진동 효과 유지
- P-02 고령 보행자의 낮은 음성 안내 볼륨과 느린 TTS 설정 유지
