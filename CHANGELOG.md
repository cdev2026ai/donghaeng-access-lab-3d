# 변경 이력

## v0.4.1 — 시작 순서 오류 수정

- 로컬 실행 직후 O-01 이벤트가 시작 화면을 덮는 오류 수정
- `시작 안내 → 이동약자 유형 선택 → 체험 시작 → 첫 이동 → O-01 이벤트` 순서 보장
- 일시정지·설정·도움말 화면 중 새 장애물 이벤트가 끼어들지 않도록 방지
- 체험 시작 후 약 0.15m 이상 이동해야 첫 장애물 이벤트가 활성화되도록 수정

# Changelog

## 0.4.0 — Event System

### Added

- 접근·상황인지·판단·동작수행·결과 5단계 이벤트 상태 머신
- 장애물 감지 반경 기반 이벤트 자동 시작
- 장애물별 안전·위험·재확인 판단 선택지
- 숫자 1~3, Enter/E 이벤트 단축키
- 시각장애 유형 F 키 상황인지 연계
- 판단 시간, 수행 시간, 재확인, 충돌, 차단 기록
- 이벤트 진행 HUD와 장애물별 상태 표시
- 장애물 통과 결과 화면
- 최종 이벤트 요약 및 장애물별 로그
- `EVENT_SYSTEM.md`
- `V0.4_TEST_CHECKLIST.md`

### Changed

- 장애물 상태를 대기·접근·인지·판단·수행·통과로 세분화

- 마지막 목적지 완료를 O-05 결과 확인 이후로 조정
- 시작·도움말·결과 화면 문구를 v0.4 흐름에 맞게 수정
