# 변경 이력

## v0.9.0 — 기획 문서 v2.0 통합 구현

### 프로젝트 목적

- 시뮬레이터를 최종 게임이 아니라 이동 보조 앱 프로젝트 전 사전 시연·요구사항 발견 도구로 재정의
- 시작 화면, 체험 안내, 결과 화면 문구를 앱 요구사항 탐색 중심으로 변경

### 연속 체험

- 장애물별 자동 모달·정답 선택·개별 결과 인터럽트 제거
- 학교 정문부터 버스정류장까지 연속 이동 흐름 적용
- 상황인지·상황판단·동작수행을 실제 행동으로 자동 기록
- 강사·개발자 설명용 분석 패널을 요청형 화면으로 분리

### 횡단보도·신호·차량

- 보행 신호 상태 머신 구현
- 횡단보도 진입·이탈 경사로와 대기선 구현
- 보행 가능 상태에서 차량 정지, 적색 상태에서 차량 이동 구현
- 위험 진입 시 차량 비상 정지 구현
- 대기·출발·횡단·신호 종료 전 완료 기록 추가
- 유형별 출발 반응·정렬 지연·속도 차이 적용

### 이동 보조 앱

- 앱 도움 없음/OFF와 앱 도움 있음/ON 조건 선택 화면 추가
- 신호·잔여 시간·예상 횡단 시간·시간 여유 계산
- 경사로·장애물·통과 조건·우회 방향 안내
- 비모달 앱 안내 패널과 선택적 TTS 추가
- 앱 I-P-O 분석 패널 추가

### 결과 화면

- 성공·판단 점수 중심 결과 제거
- 신호 대기·출발·횡단·경로 조정·차단 중심 관찰 결과로 변경
- 부족했던 정보와 이동 보조 앱 요구사항 자동 요약
- 성찰·교차 비교·결과 저장·내보내기 제외 유지

### 유지 기능

- 장애물 O-01~O-05
- 이동약자 P-00~P-03
- 피로도·불안감·방향 확신도·시간 압박 상태 UI
- 시각장애인 시야 효과와 보이는 흰지팡이
- 설정·도움말·저사양·고대비 기능
