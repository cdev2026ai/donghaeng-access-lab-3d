# 변경 이력

## v0.6.2 — 결과 화면 버튼 레이아웃 정리

- `다른 유형 체험` 버튼을 결과 카드 전체 너비의 대표 행동으로 배치
- `같은 유형 다시 체험`과 `시작 화면`을 동일 너비 2열로 배치
- 세 버튼의 높이·모서리·글자 굵기와 간격을 통일
- 620px 이하 화면에서는 보조 버튼을 1열로 전환
- 버튼 영역의 하단 여백과 대표 버튼 그림자를 조정

## v0.6.1 — 수업용 미션·결과 화면 간소화

- 미션 브리핑과 실시간 미션 체크리스트 유지
- 이동 기록·상태 지표·장애물별 이벤트 결과 화면 유지
- 워크북과 중복되는 성찰 입력 화면 제거
- 팀 활동과 중복되는 교차 체험 비교 화면 제거
- 브라우저 기록 저장과 localStorage 사용 제거
- JSON·CSV 및 비교 CSV 내보내기 제거
- 결과 화면 버튼을 다른 유형 체험·같은 유형 재체험·시작 화면으로 단순화
- 결과 안내 문구에서 자동 저장·비교 관련 표현 제거

## v0.6.0 — 미션·결과·비교 화면

- 이동약자 유형 선택 후 미션 브리핑 화면 추가
- 체험 전 경로, 목표, 유형별 관찰 포인트 표시
- 실시간 이벤트·안전 판단·목적지 체크리스트 추가
- 결과 화면을 경로·이벤트·판단·재시도·상태 지표 중심으로 확장
- 체험 기록 기반 자동 해석 문구 추가
- 성찰 질문 3종과 브라우저 저장 기능 추가
- 완료 기록 최대 8회 localStorage 저장
- 2~4개 기록 교차 비교 화면 추가
- 개별 JSON/CSV와 비교 CSV 내보내기 추가
- 사람의 능력을 순위화하지 않는 교육적 비교 원칙 반영

## v0.5.2 — 시야 제한 효과 추가 강화

- 기본 `중간` 강도의 주변부 마스크와 중심 시야 축소를 한 단계 강화
- 시야 명료도 기본값과 장애물 구간별 목표값을 하향 조정
- O-04 점자블록 단절 구간에서 시야 부담이 더욱 분명하게 나타나도록 조정
- 장면 블러·채도 저하·밝기 저하를 소폭 강화
- 중앙과 주변부의 경계가 더 빠르게 어두워지도록 비네트 그라디언트 개선
- 완전한 암전은 피하고 중앙 가용 시야와 흰지팡이 탐지는 유지

## v0.5.1 — 시각장애인 시야 체험 효과 강화

- 기본 시야 효과 강도를 `낮음`에서 `중간`으로 변경
- 시야 효과 `강함` 옵션 추가
- 중앙 가용 시야를 축소하고 주변부 암화·터널 시야를 강화
- 시야 명료도에 따라 장면 블러·채도·밝기가 함께 변화하도록 개선
- O-03·O-04·O-05에서 시야 정보 부담이 더 뚜렷하게 나타나도록 조정
- 저사양 모드에서도 주변부 마스크는 유지
- 완전한 암전 대신 정보 접근 제한을 표현하도록 유지

## v0.5.0 — 체험 효과와 상태 UI

- 피로도·불안감·방향 확신도·시간 압박 상태 UI 추가
- 이동·경사·거친 노면·충돌·판단 선택 기반 상태 계산 추가
- 시각장애인 주변부 시야 제한과 정보 명료도 효과 추가
- 피로 헤이즈와 불안 맥동 효과 추가
- 고령 보행자 피로 누적, 횡단보도 시간 압박 반영
- 시각장애인 1인칭 흰지팡이 3D 모델과 F 키 애니메이션 추가
- 체험 효과, 시야 강도, 고대비 UI, 흰지팡이 표시 설정 추가
- 최종 결과에 상태 요약 추가
- `EXPERIENCE_EFFECTS.md`, `V0.5_TEST_CHECKLIST.md` 추가
